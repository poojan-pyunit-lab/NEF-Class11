# Class 11 C Programming Solutions - Xavier Academy (NEF)

Welcome to my repository of **Class 11 C Programming solutions** for Xavier Academy, following the NEF syllabus.  

## Contents

### 1️⃣ Basics of C Programming
- `hello_world.c` - Display "Hello World"
- `sum_two_numbers.c` - Read two numbers and find their sum
- `quotient_remainder.c` - Divide integers and find quotient & remainder
- `circle_area_perimeter.c` - Compute area and perimeter of a circle
- `calculate_y.c` - Program to calculate y
- `student_info.c` - Read student info and display in proper format

### 2️⃣ Conditional Statements
- `greatest_of_three.c` - Find greatest among 3 numbers
- `check_leap_year.c` - Check leap year
- `odd_or_even.c` - Determine odd or even number
- `divisible_by_3_not_5.c` - Display number divisible by 3 but not by 5
- `prime_or_composite.c` - Determine prime or composite number
